# Автостарт

Игра поддерживает автоматический запуск с выбранной локацией через параметр командной строки.

## Использование

### В редакторе Godot:
```bash
godot -- --autostart 0
```

### Экспортированная версия:
```bash
./osm-racing -- --autostart 0
```

## Индексы локаций

- `0` - Череповец (по умолчанию)
- `1` - Москва (Отрадное)
- `2` - Тбилиси (Важа-Пшавела)
- `3` - Дубай (Крик)

## Примеры

Запуск в Москве:
```bash
godot -- --autostart 1
```

Запуск в Тбилиси:
```bash
./osm-racing -- --autostart 2
```

## Отладка

При запуске с автостартом в консоли будут выведены debug сообщения:
- Список пользовательских аргументов
- Выбранная локация
- Ход загрузки террейна

Если загрузка зависает, проверьте логи на предмет ошибок.
